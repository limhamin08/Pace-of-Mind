# Pace-of-Mind

👣 마음의 보폭 (Pace of Mind)
내 마음의 속도에 맞춘 AI 협업 학습 가이드

마음의 보폭은 사용자의 MBTI 성향을 분석하여 최적의 학습 페이스를 제안하고, AI(Grok-3)를 통해 복잡한 CS 개념을 시각적 도식으로 변환해주는 지능형 학습 트래커입니다.

✨ 주요 기능
AI 개념 도식화 (Mind Visualization): 학습 키워드를 입력하면 Grok-3가 분석하고 Grok-Imagine이 시각적 도식을 생성하여 직관적인 이해를 돕습니다.

MBTI 맞춤형 피드백: 사용자의 성향에 따라 '힐링형' 또는 '성취형' 학습 피드백을 제공합니다.

인터랙티브 퀴즈: AI가 생성한 도식 카드를 활용해 학습 내용을 복습하고 성취도를 기록합니다.

학습 히스토리 관리: 로컬 스토리지를 활용해 개인별 학습 보폭과 성장을 추적합니다.

🛠 기술 스택
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

AI Integration:

xAI Grok-3: 텍스트 분석 및 프롬프트 엔지니어링

xAI Grok-Imagine: CS 교육용 도식 이미지 생성

Pollinations AI: 이미지 생성 서버 우회 및 백업

Storage: Browser LocalStorage
